## Overview

FlowKit provides the building blocks for creating interactive node-based experiences in React.

Whether you're building a workflow engine, automation designer, network topology editor, ETL pipeline, or visual scripting system, FlowKit offers a flexible foundation with customizable nodes, edges, and interactions.

---

## Current Features

### Canvas

* Infinite workspace
* Pan and zoom controls
* Grid background
* Minimap
* Smooth navigation

### Nodes

* Drag-and-drop positioning
* Custom React node components
* Input and output endpoints
* Node selection
* Group containers

### Edges

* Bezier connections
* Orthogonal connections
* Custom edge rendering
* Edge selection
* Animated flow paths

### Interactions

* Create connections between endpoints
* Move and organize nodes
* Selection management
* Extensible event handling

### Developer Experience

* Built with React and TypeScript
* Fully customizable rendering
* Lightweight architecture
* Zustand-based state management
* Strong typing throughout
